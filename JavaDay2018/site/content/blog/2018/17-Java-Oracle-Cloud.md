Desplegando aplicaciones Java en Oracle Cloud

## Social Media

## Description

Una típica aplicación Java requiere un servidor de aplicaciones y una base de datos, Oracle Cloud tiene servicios como DB y Java Cloud  listos para desplegar una aplicación Java sin un mayor esfuerzo. Esta sesión a partir de un demo presenta desde el proceso de aprovisionamiento, hasta el despliegue.

Los web services RESTFul son muy populares hoy día, estos son construidos para trabajar mejor en internet, su uso permite compartir datos entre sistemas. Por otro lado, la computación en la nube (cloud computing) es cómputo basado en internet, se puede asumir que la mejor tecnología para desplegar servicios RESTFul es el Cloud. Algunas APIs de Java que pueden ser usados para convertir cualquier método en un RESTFul service simplemente adicionando anotaciones al código. Esta sesión presentada por medio de un demo presenta como usando anotaciones Java se puede crear un RESTFul service el cual es publicado utilizando Oracle Cloud para almacenar y publicar datos.
