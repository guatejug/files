## Title

Introducción a Processing

## Presenter

Aldo Ebany Pérez Larios

## Description

Processing es un libro de bocetos para codificar en el contexto de las artes visuales, desarrollado por artistas y para artistas; Es un lenguaje de programación basado en Java, multiplaforma, escalable, robusto y ademas sencillo en su aprendizaje.

Processing es un lenguaje de programación diseñado para las artes visuales y dirigido a personas no programadores, esto por su sencillez de aprendizaje, fácil utilización e instalación, pero esto no lo hace limitado ya que es posible hacer programas interactivos con salida 2D, 3D, PDF o SVG, ademas de tener integración con OpenGL para 2D acelerado y 3D. Esta basado en Java por lo que hereda mucho de su funcionalidad, ademas de ser multiplataforma ya que puede ser utilizado en GNU/Linux, Mac OS X, Windows, Android y ARM.
