## Description

En otoño de 2017, Oracle anunció  oficialmente la donación de la plataforma Java EE a la Fundación Eclipse bajo el proyecto sombrilla llamado Eclipse Enterprise para Java (EE4J). Como resultado, hay grandes cambios para el entorno Java empresarial y su comunidad. Hay un nuevo proceso de estandarización, un nuevo conjunto de especificaciones en proceso y una nueva marca para la plataforma Java EE, Jakarta EE. Estos son los cambios más significatios que la plataforma Java EE ha tenido desde su presentación al mundo hace  ya casi 20 años. Durante esta sesión podrás obtener con detalle esta información de la mano de involucrados directos en esta nueva etapa para la plataforma Java Empresarial.
