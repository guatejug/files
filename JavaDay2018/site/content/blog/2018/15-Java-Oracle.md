## Presenter

## Social Media

## Description

Oracle provee la capacidad de incrustar código Java y este ser ejecutado desde PLSql, dicha capacidad abre un mundo de posibilidades para solucionar necesidades  del negocio de manera creativa, eficiente y con una excelente velocidad de respuesta.

Oracle Database provee soporte para desarrollar, almacenar e implementar código de aplicaciones java, en 45 minutos se presentara el lenguaje Java para los desarrolladores de PLSql de Oracle que ya cuentan con funciones y procedimientos, y como con Java pueden potenciar e implementar soluciones que aprovechen la escalabilidad y rendimiento de Oracle Database.


## Bio(s)

Gerber ha trabajado con tecnología Oracle en Centro América por más de 9 años en varios sectores como Telecomunicaciones, Banca, Servicios Financieros, Aseguradoras e Instituciones Gubernamentales.

Su especialización es Oracle Database, implementando proyectos utilizando Oracle Rac, DataGuard, Golden Gate; Atendiendo problemas de desempeño, Configuraciones de Sitios de desastre, diseñando arquitecturas de bases de datos, migraciones y actualización de versión.
En 2013 Gerber participo en varios proyectos implementando Oracle Exadata en Guatemala, iniciando desde el apoyo en las pruebas de concepto como parte de los esfuerzos de preventa, hasta las migraciones de bases de datos dentro del Exadata.

Gerber es un entusiasta de la tecnología y ha explorado distintas servicios de Cloud; diseñando, implementando y migrando infraestructuras a Amazon AWS y Oracle Public Cloud; También cuenta con cierta experiencia trabajando con Middleware de Oracle Glassfish, Tomcat, Apache Http Server y aplicaciones con Oracle Apex.

En 2017 Gerber se une al equipo de Nuvola Consulting Group, una Startup guatemalteca de tecnología con especialización en Tecnologías Oracle y en 2018 ingresa al equipo de eProseed de América Central.
