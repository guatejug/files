## Description

La conferencia la quiero orientar mas que todo para inspirar a los participantes a ser creativos para solventar problemas y ayudar a las empresas a confiar en el poder que las herramientas de Java EE / OSS proporcionan.

Expondre en pocos minutos los motivos por los cuales Java EE ayuda a las empresas con sus sistemas en 3 grandes Ambitos:

* Creación de nuevos Sistemas
* Integración de Sistemas
* Lidiar con el caos

Usaré de ejemplo casos reales donde Java EE y El OSS solucionó con éxito problemas de negocio en el sector Público y Privado :

* El Sistema de Pagos (Como JavaEE/Glassfish/MySQL crearon un gateway de pagos en supermercados y Farmacias de El Salvador)

* El Sistema Judicial (Historia de una mala implementación de un proyecto Java EE y su redención)

* Los sistemas del Banco Central (Como JavaEE/Springframework y la capacitación ayudaron a los sistemas del Banco Central )

* El Sistema de Conocimiento y Aprendizaje (Como creando tu Propio Runtime de Coldfusion con Tomcat logro mejorar un Legacy System)

* La migración al sistema Millonario (Cuando tu Nuevo sistema que costo millones no esta personalizado para tu Negocio, Wildfly y Java EE al rescate)
