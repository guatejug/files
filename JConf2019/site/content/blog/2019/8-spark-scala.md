## Description

En esta charla se proporcionara una introducción a la programación funcional de Scala con Spark, con un ejemplo práctico.

La programación funcional (FP) en Scala es un estilo de desarrollo de software que enfatiza las funciones para que no dependan del estado del programa. El código funcional es más fácil de probar y reutilizar, más sencillo de paralelizar y menos propenso a errores que otros códigos.

Por otro lado Apache Spark es un sistema de computación en clúster rápido y de propósito general. Proporciona API de alto nivel en Java, Scala, Python y R, y un motor optimizado que admite gráficos de ejecución generales. También es compatible con un amplio conjunto de herramientas de alto nivel que incluyen Spark SQL para SQL y procesamiento de datos estructurado, MLlib para aprendizaje automático, GraphX ​​para procesamiento de gráficos y Spark Streaming.

En esta sección se explicara la combinación de SCALA con SPARK como hacen buen equipo para el procesamiento y análisis de datos.

Además, trae el portátil para que tenga la oportunidad de probar Scala. Asegúrate de traer instalado el entorno de ejecución (http://www.scala-lang.org/download/install.html).
