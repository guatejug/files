## Description

Para esta charla se necesita un entendimiento medio del lenguaje Java, además se debe tener algo de experiencia en la construcción de aplicaciones web incluyendo configuración de conectividad con otros servicios por ejemplo base de datos

La expansión de comercio electrónico ha sido constante, la detección de fraudes se ha vuelto un factor clave y es necesario reaccionar a tiempo. En esta charla vamos a implementar una solución de BigData que permita procesar y analizar la información en tiempo real haciendo uso de Data Streaming.

Las ventas en línea y la banca digital han crecido constantemente los últimos años gracias a la confianza de los clientes en los pagos electrónicos, esto vuelve la detección de fraudes un factor crítico. Dicho proceso demanda reaccionar en tiempo (casi) real aplicando técnicas de diseño e implementación de soluciones escalables que combinen técnicas de Machine Learning con la ingestión y análisis de cantidades masivas de transmisión de datos(Fast Data Streaming).

En esta charla vamos a construir una aplicación web en Java que interactúe con un servicio de detección de fraudes. La idea general es diseñar una micro arquitectura con la capacidad de manejar conceptos y tecnologías como BigData, Data Streaming, Data Analysis y Machine learning.
