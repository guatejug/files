## Title

¿Como desarrollar una aplicación Android con Web Services y no morir en el intento?

## Presenter

**Max Alejandro Cerna Flores**

Max Cerna, Ingeniero en Ciencias y Sistemas, Java developer para aplicaciones Web y Mobiles, asi como DBA en tecnologias Oracle, miembro colaborador del Grupo de Usuarios Java en Guatemala, consultor de Base de Datos en Nabenik y actual DBA en Atos IT Technologies Guatemala S.A.


## Description

Uno de los mayores retos que tiene toda persona que empieza en el mundo de la programación para dispositivos móviles con Android es, como almaceno la información de mi aplicación en una base de datos más robusta fuera del dispositivo, para esta presentación, se pretende desarrollar una aplicación móvil en Android la cual deberá conectarse a una base de datos MySQL mediante Restful Web Services.

Durante la presentación se cubrirán puntos como: 1) Como conectarse a una base de datos en Java utilizando JPA. 2) Como crear y publicar un Restful WebServices en WildFly (WebServer) 3) Como desarrollar una aplicación Android simple con listas. 4) Como consumir un Restful WebServices desde nuestra aplicación Android.
