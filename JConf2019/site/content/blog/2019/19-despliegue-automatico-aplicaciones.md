## Description

¿Cuánto tiempo necesitas para desplegar tu aplicación? ¿Qué pasos necesitas seguir para hacerlo? Podrás conocer como preparar tu aplicación y tu entorno de desarrollo para poder automatizar estas tareas y dejar el estrés en el olvido.

Imagina la posibilidad de desplegar una aplicación en un entorno de producción de forma automática, mediante un click de un botón o a través de un commit en el control de versiones del proyecto.

Imagina que ya no necesitas decirle a tus usuarios que se salgan de tu aplicación por 5 minutos, que durante ese período de tiempo no tendrás que: - Compilar tu aplicación con las credenciales correctas para el entorno de producción - Parar el servidor de aplicaciones o replegar la aplicación - Copiar los archivos ejecutables al servidor desde tu computadora - Desplegar tu aplicación o reiniciar el servidor

Afortunadamente puedes hacer esto y más a través de una receta de despliegue utilizando el software Jenkins.
